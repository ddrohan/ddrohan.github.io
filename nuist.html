# Object Oriented Design

Object Oriented Design (OOD) is the process of planning a system of interacting objects for the purpose of solving a software problem. Here we look at OOD in detail and apply the principles of this design to build a moderately complex application.
# GUI Development

Android applications use classes in the Java and Kotlin language to implement various aspects of Graphical User Interface (GUI) functionality. Here we investigate GUI development through through the medium of the Android OS.
